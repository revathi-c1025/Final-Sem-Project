Git Operations

Manage version control for your project. Configure your GitHub token in the Configuration page to enable push operations.

ℹ️ Setup Required

To use push operations, you need to:

  1. Generate a GitHub Personal Access Token (PAT) from https://github.com/settings/tokens
  2. Go to Configuration page → Git Configuration section
  3. Paste your token in the "GitHub Personal Access Token" field
  4. Click "Save Changes"

Branch Name

If the branch doesn't exist locally, it will be created automatically when you push.
